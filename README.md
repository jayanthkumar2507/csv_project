# Resume Analyzer

A Flask-based web application that analyzes resumes and extracts useful information.

## Features
- Upload resume files (PDF)
- Analyze resume content
- Display extracted details in a clean UI

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Google Gemini 

## Project Structure
- app.py – Main Flask application
- templates/ – HTML files
- static/ – CSS files
- resumes/ – Uploaded resumes (local use)

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the Flask app using:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/

## Note
API keys and virtual environments are ignored using `.gitignore`.
